Proyectos destacados

Landing Page para Evento

Una landing page completamente responsive, desarrollada en HTML y CSS siguiendo metodologías como BEM e ITCSS. Incluye modo oscuro, formulario de registro, carrusel en CSS y está pensada para ser accesible y funcional en cualquier dispositivo.

Gestión de Inventario en Python

Un sistema de consola creado en Python que permite registrar productos, actualizar información, eliminar y calcular el valor total del inventario. Utiliza estructuras como listas, diccionarios, funciones y manejo de datos para una gestión eficiente.

Calculadora Financiera

Una herramienta práctica para ayudar a los usuarios a calcular cuánto pueden ahorrar según sus ingresos y gastos mensuales. Desarrollada con una interfaz web clara y amigable, ideal para principiantes en el mundo de las finanzas personales.
